RoadWorks Create Addon

About Create: Roadworks

Create: Roadworks is a Minecraft mod addon for the Create mod, themed around road construction, utility infrastructure, and hi-rail technology. Inspired by real-world construction and maintenance equipment, the goal of this project is to expand the Create experience with new mechanics and decorative elements that reflect industrial roadside work.

Mod Features (Planned):

  • Hi-rail vehicles – Trucks and machinery that can drive on roads and rails
  • Road construction tools – Signs, cones, barriers, and equipment
  • Utility infrastructure – Functional and decorative blocks like cabinets, poles, and maintenance gear
  • New transport systems – Concepts for moving materials across mixed road-rail networks

Development Status:

The project is in early development. While the base mod structure exists and several Blockbench models have been created, there are currently no implemented in-game features. Recent restructuring of the team means progress is temporarily paused while roles are refilled.

Technical Info:

  • Built for Minecraft Java Edition
  • Designed for the Create mod (Forge/Fabric/Architectury support being explored)
  • Art style and modeling follow Create’s clean, semi-realistic industrial look

Future Goals:

  • Introduce working hi-rail vehicles using Create’s mechanics
  • Expand the set of construction-themed blocks and tools
  • Establish compatibility with major transport-related Create addons

This mod is still in its foundational phase and is being developed as a passion project by the community. Updates and development progress will be shared on the official Discord server.
